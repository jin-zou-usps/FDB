# FDB - SQL Query Collection

A comprehensive collection of SQL queries organized by category and complexity level. This repository serves as a reference and learning resource for SQL developers, database administrators, and data analysts.

## 🗂️ Collection Overview

This repository contains over 100+ carefully crafted SQL queries organized into the following categories:

### 📚 Basic Queries (`sql-queries/basic/`)
- **SELECT Queries**: Fundamental data retrieval operations
- **JOIN Operations**: All types of table joins with examples
- **Aggregation Functions**: GROUP BY, HAVING, and statistical functions

### 🔧 Data Manipulation (`sql-queries/dml/`)
- **INSERT Operations**: Data insertion patterns and techniques
- **UPDATE Queries**: Data modification with various conditions
- **DELETE Operations**: Safe data removal practices

### 🏗️ Database Structure (`sql-queries/ddl/`)
- **Table Creation**: Schema design with constraints and indexes
- **Table Modifications**: ALTER TABLE operations and schema changes
- **Object Management**: Database objects creation and removal

### 🚀 Advanced Techniques (`sql-queries/advanced/`)
- **Subqueries**: Scalar, correlated, and nested query patterns
- **Window Functions**: Analytical functions for complex calculations
- **Common Table Expressions (CTEs)**: Including recursive CTEs

### 🛠️ Utilities & Administration (`sql-queries/utilities/`)
- **Database Administration**: Monitoring, maintenance, and DBA tasks
- **Performance Analysis**: Query optimization and troubleshooting
- **Data Analysis**: Statistical analysis and business intelligence queries

## 🚀 Quick Start
